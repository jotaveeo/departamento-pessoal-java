# Cadastro de Funcionários e Dependentes

Este projeto implementa um sistema simples de cadastro de funcionários e seus dependentes, com funcionalidades de cadastro, cálculo de bônus mensal, exclusão e alteração de salário.

## Funcionalidades

1. **Cadastrar Funcionário**: Permite cadastrar um novo funcionário e seus dependentes.
2. **Mostrar Bônus Mensal**: Exibe o bônus mensal de cada funcionário com base no número de dependentes.
3. **Excluir Funcionário**: Exclui um funcionário e seus dependentes da lista.
4. **Alterar Salário de Funcionário**: Permite alterar o salário de um funcionário cadastrado.

## Como Usar

1. Compile e execute os arquivos Java fornecidos.
2. O menu interativo aparecerá, onde você poderá escolher as opções disponíveis.
3. As opções permitirão cadastrar funcionários, alterar salários, excluir funcionários e exibir os bônus mensais.

## Estrutura do Projeto

- **Funcionario.java**: Classe que representa um funcionário.
- **Dependente.java**: Classe que representa um dependente de um funcionário.
- **CadastroFuncionario.java**: Classe responsável pela lógica de cadastro, alteração e exclusão de funcionários.
- **Main.java**: Classe principal que interage com o usuário e chama as funcionalidades de cadastro.
